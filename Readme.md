# Lab 03 & 04 – CSS Fundamentals

In this lab, different CSS implementation methods were practiced including inline, internal, and external CSS. Various selectors such as element, class, ID, universal, and attribute selectors were used to style HTML elements.

CSS combinators and pseudo-classes were applied to understand relationships between elements and to style elements based on their states like hover, focus, and active. Pseudo-elements were also used to style specific parts of text and add generated content.

Typography styling was implemented using different font families, including Google Fonts. Text properties such as alignment, decoration, transformation, spacing, and shadow effects were applied to improve readability and design.

Background styling techniques were demonstrated using solid colors, background images, gradients, and multiple background layers. Image styling was also performed by applying borders, rounded corners, shadows, filters, and hover effects.

The CSS box model was implemented using margin, padding, borders, and box-sizing. Transformations such as translate, rotate, scale, and skew were applied, along with transitions to create smooth visual effects.

Different UI components were styled including buttons, forms, cards, and navigation menus with hover effects, dropdowns, and responsive design elements. Advanced CSS effects like gradient text, loading animations, and custom form controls were also demonstrated.
